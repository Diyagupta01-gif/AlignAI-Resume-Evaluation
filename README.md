# AlignAI – Resume Requirement Evaluation System

AlignAI is a KPI-driven Resume Evaluation Framework designed to assess candidate-job alignment using AI-powered analysis and structured scoring.

## 🔍 Features

- AI-powered resume analysis
- Skill gap identification
- KPI-based scoring system
- Risk classification (Low / Moderate / High)
- Interactive dashboard
- Downloadable stakeholder report

## 🛠 Tech Stack

- Python
- Streamlit
- Google Gemini API
- Pandas
- Matplotlib
- PDFPlumber

## 🚀 How It Works

1. Upload a resume (PDF)
2. Paste job description
3. System extracts skills
4. Calculates alignment score
5. Displays skill gap analysis
6. Generates downloadable report

## 🎯 Use Case

Designed as a decision-support tool for recruiters and hiring managers to evaluate candidate-job fit using measurable metrics.
